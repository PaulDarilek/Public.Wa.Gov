# Public.Wa.Gov

Public tools and packages useful for Washington State Gov data



All code contributions must be public domain



## Purpose:

* Build a public repository useful for Washington State (USA) agencies to freely use or add to.
* Code should be related to Washington state data or formats that are publicly accessible:
* Because these are open source (public domain) and accessible by the public, they cannot be considered Category 3 because of security concerns.
* Goal is to save taxpayers money because this code can be reused by any agency (subject to agencies policy about using public domain code).



### Examples include:

* Parsers for publicly downloadable files (Category 1) that can be downloaded.
* Tools to help with File Formats that are shown publicly by \*.wa.gov (whether or not the data is publicly downloadable).



### Examples to exclude:

* General use utilities such as Image tools that may split or join TIFF images (contribute to another public repository).
* Security related code specific for wa.gov  (security by obscurity is not real security, but just don't do it)
